How does Javascript work?


We know Javascript is a interpreted language.An Interpreter takes the Javascript code and runs it.Javascript codes are executed from top to bottom.In browser,we need Javascript engine to run the code.Javascript engine compiles the code and this code is also known as bytecode.Interpreter takes the bytecode and run this through machine.

What is the difference between session storage and local storage?


  • When page session ends,localStorage data is saved but sessionStorage data is cleared.
  • If we open a new tab,localStorage will be still available but sessionStorage will expire.
  • Local storage has 5mb limit but sessionStorage has 5-10mb limit.
  • LocalStorage is available for all current and future visits to the sites but sessionStorage is only available when window is opened.
  • In localStorage we have to manually delete data but in sessionStorage when the window is closed data is expired.

    • What is event loop in Javascript?


    In javascript runtime model event loop is important.The event loop constantly monitors call stack and event queue.When call stack is empty,it takes the first event of event queue and pushes it to call stack.If it is not empty ,current event will be processed.Event loop repeat the process untill all tasks is finished.
